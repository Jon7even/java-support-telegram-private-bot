# SupportBot v1.0
> Всем привет! Перед вами небольшой бот-помощник для оптово-розничной компании.

> [!NOTE]
> На данный момент бот находится в разработке, поэтому информация 
> по функционалу и развертыванию будет появляться постепенно.

> [!IMPORTANT]
> Создано с помощью:
- Java 11
- Telegram Bots
- RabbitMQ
- Apache Maven
- Spring boot 2.7.14
- Hibernate
- PostgreSQL
- Liquibase
- Lombok
